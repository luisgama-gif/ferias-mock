# Férias Mock - Solicitação de Férias no Pópulis

Interface mockada para solicitação de férias no sistema Pópulis

## Como configurar o ambiente

1. **Criar ambiente virtual**

```powershell
python -m venv venv
```

2. **Ativar ambiente virtual**

```powershell
venv/Scripts/activate
```

## Como rodar

1. **Instalar dependências:**

```powershell
npm install
```

2. **Executar o projeto:**

```powershell
npm run dev
```

3. **Acessar no navegador:**

```
http://localhost:3000
```
